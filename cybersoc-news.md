## CTF Learning Resources
We're well aware that **lots of you want to learn to hacking** and how to 
**compete in CTFs**, but that starting off can be daunting. We've published a 
lot of learning resources on our GitHub recently, and I wanted to make sure 
you know where they are :)

### IntakeCTF Challenges and Writeups
- All of the challenges from **IntakeCTF** have full writeups.
- We have a GitHub repository with all of the files used to create the 
challenges, plus those writeups.
- You can find it **[here](https://github.com/WarwickCyberSoc/intakectf-2025)**


### IntroToCTF Class Resources
- We've created a GitHub repository with **slides** and **class resources** from
our Monday **IntroToCTF** series.
- We'll be updating it each week when we cover a new beginner CTF topic.
- You can find it **[here](https://github.com/WarwickCyberSoc/intro-to-ctf)**
